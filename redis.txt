切换到/redis/src目录下
	启动：./redis-server
	关闭：./redis-cli -h 127.0.0.1 -p 6379 shutdown
	验证方式关闭：./redis-cli -h 127.0.0.1 -p 6379 -a 123456 shutdown
	操作：./redis-cli
	验证方式操作：./redis-cli -h 127.0.0.1 -p 6379 -a 123456
	
远程连接
	sudo gedit ~/redis/redis.conf
	将bind 127.0.0.1注释掉并将protected-mode设置为no
	找到requirepass设置连接验证密码（必须）
	127.0.0.1:6379> config set requirepass 123456 配置连接验证（必须）这种方式每次启动都要配置
